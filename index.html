Testing the Clarity ↔ UET integration

A small static site that loads Microsoft Clarity through Google Tag Manager against the Clarity staging environment. The site intentionally ships no hand-rolled UET so any bat.bing.com traffic you observe is attributable to the Clarity integration.

How it works • Contact

Tip: open DevTools → Network and filter on clarity and bat.bing.com to watch the integration light up.

1. Page load

GTM bootstraps and injects the Clarity tag against the staging container.

2. Clarity captures

Clarity begins recording session data against the configured staging project.

3. UET injected

If MsAds linking is configured for the project, Clarity drops the UET tag — verify via bat.bing.com.
