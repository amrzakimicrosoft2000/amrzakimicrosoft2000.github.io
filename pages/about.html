About this site

This is a thin static host used to validate the integration between Microsoft Clarity and UET (Universal Event Tracking). Clarity is loaded through Google Tag Manager against the Clarity staging environment.

What it tests

When MsAds linking is enabled on the Clarity project, Clarity is expected to inject the UET tag into the host page. The site intentionally ships no hand-rolled UET so any bat.bing.com requests you see are attributable to the Clarity integration.

Files to know

Styles: assets/css/styles.css

JS: assets/js/site.js

Pages: pages/*.html

Clarity via GTM

Every page embeds the standard GTM snippet (GTM-PC77ND9V). Configure the container to load Clarity from the staging tag host.

Staging endpoint

Clarity should be configured to point at https://claritydashboard.asgfalcon-staging.io.

Verifying UET

Watch DevTools Network for bat.bing.com requests — that's the injected UET tag firing.
