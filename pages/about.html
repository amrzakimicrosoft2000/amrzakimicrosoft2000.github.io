About this website

This is a tiny static site used for testing GitHub Pages and basic navigation across multiple pages.

What’s included

A shared header/footer, responsive layout, theme toggle, and example content pages. Everything is plain HTML/CSS/JS so it’s easy to tweak.

Files to know

Styles: assets/css/styles.css

JS: assets/js/site.js

Pages: pages/*.html

Why static?

It’s the simplest way to validate hosting, caching, and routing behavior.

How to extend

Add more pages under pages/ and link them in the nav.

SEO basics

Add robots.txt and sitemap.xml if you want to test crawlers.
